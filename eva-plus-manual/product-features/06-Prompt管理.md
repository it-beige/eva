# Prompt管理

---

## 一、创建编写Prompt

1. 在左侧导航栏，选择 Prompt 工程 > Prompt 管理。
2. 在打开的 Prompt 管理页面，单击 + 新建Prompt。
3. 弹窗内可以编写自己的Prompt
4. 点击列表中的prompt，弹出的新页面即 Prompt详情页，可以支持查看Prompt的具体内容、版本情况。
5. 点击 Promptground运行，可以跳转到Playground进行相应调试

## 二、调试Prompt

1. 在Playground中选择对应的Prompt，以及评测集，单击 RUN，即可运行查看Prompt运行结果
2. Prompt设置的右上角可以选择对应的模型及参数
3. 如果需要多个Prompt对比，可以点击 添加Prompt，即可基于同一评测集同时测试多个Prompt输出效果
4. 在Playground中如果输出结果不理想，可以在线修改并保存为新版本
5. 调试过程会以评测任务的形式记录在评测任务管理列表，可以随时回溯查看
