# 评测集: AI数据加工

---

在评测集详情页面，选择 AI数据加工 可以通过写prompt基于已有数据列生成新的数据列

AI数据加工能力可用于非常多场景，这里为你提供了丰富多样的场景数据加工prompt灵感

## AI数据加工「prompt灵感库」

### Prompt设计最佳实践

- 明确输入输出：清楚标注 ${输入列} 和 &{输出列}
- 结构化指令：使用分点、分步骤的方式组织prompt
- 明确标准：给出评判标准或分类依据
- 控制输出格式：明确要求输出格式（纯文本/JSON/数字等）
- 示例引导：必要时提供输出示例

### Agent分类（基于评测数据结构特征）：

1. 对话交互类 Agent：单轮/多轮问答，客服助手、选品推荐
2. 任务执行类 Agent：工具调用（如 API 执行），流程自动化
3. 内容生成类 Agent：创作型内容（如文档生成），代码生成

## 1. 对话交互类Agent：评测集 AI数据加工「prompt灵感」

### 对话交互类Agent的评测数据，有哪些二次加工需求（即AI数据加工prompt分类）：

- 内容提取（从现有数据中提取信息）
- 内容转换（改变数据的形式或结构）
- 内容生成（基于现有数据生成新内容）
- 内容分类（对数据内容进行分类打标）
- 质量打标（对数据质量进行分类打标）
- 情感打标（对文本情感进行分类打标）

### 如何选择合适的prompt：

| 评测目的     | 推荐Prompt类型      |
| ------------ | ------------------- |
| 模型效果评估 | 质量分类 + 标签分类 |
| 数据集清洗   | 内容提取 + 质量分类 |
| 训练数据准备 | 内容生成 + 内容转换 |
| 问题分析     | 内容提取 + 标签分类 |
| 数据增强     | 内容生成 + 内容转换 |

### 1.6 综合应用案例

**step1.原始的评测数据**  
当你的评测集中只有input和output时  
用户输入：${user_input} ，Agent回复：${agent_output}

**step2.AI的第一层加工**  
可以通过input和output加工出：  
业务场景分类 {business_scenario}、问题难度等级 {difficulty_level}、对话质量等级 {quality_grade}

**step3.AI的第二层加工**  
通过业务场景分类、问题难度等级、对话质量等级，可以输出每条数据的综合价值评估，从而挑选出优质数据集
