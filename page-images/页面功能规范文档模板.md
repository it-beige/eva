## 📋 文档使用说明

### 如何创建新的页面规范文档

1. **复制本模板**: 将本文档复制到对应页面的子目录下，重命名为 `页面功能规范文档.md`
2. **替换占位符**: 搜索并替换所有 `[方括号]` 中的占位符内容
3. **补充截图信息**: 在对应章节中引用实际的截图文件
4. **调整内容**: 根据具体页面的特点，增删或修改章节内容
5. **更新元信息**: 修改文档版本、日期、适用范围等元信息

### 占位符说明

| 占位符             | 说明                     | 示例                                |
| ------------------ | ------------------------ | ----------------------------------- |
| `[页面模块名称]`   | 页面的功能模块名称       | 项目列表、评测集管理、应用分析      |
| `[模块名称]`       | 模块的中文名称           | 项目管理、评测管理、数据分析        |
| `[Module Name]`    | 模块的英文名称           | Project Management、Eval Management |
| `[页面子目录名称]` | page-images 下的子目录名 | 项目列表、评测集、应用分析          |
| `YYYY-MM-DD`       | 文档创建日期             | 2026-04-17                          |

### 快速开始

```bash
# 1. 创建页面子目录
mkdir -p page-images/[页面子目录名称]

# 2. 将截图放入子目录
cp [截图文件] page-images/[页面子目录名称]/

# 3. 复制并修改本文档
cp page-images/页面功能规范文档.md page-images/[页面子目录名称]/页面功能规范文档.md

# 4. 编辑文档，替换占位符
vim page-images/[页面子目录名称]/页面功能规范文档.md
```
